# BitePair

**BitePair** is a web-based food and drink ordering application that enhances the dining experience with AI-powered pairing recommendations and safety features for customers with food allergies.

## ✨ Features

- 🧠 **AI Pairing Suggestions**  
  Uses OpenAI to recommend food and drink pairings based on your selection—choose a meal, get a perfect drink suggestion (and vice versa).

- ⚠️ **Allergen Awareness**  
  Clearly labeled allergens on all menu items with interactive tags. Users can explore ingredient details and discover related items that contain similar allergens, improving food choice awareness and safety.

- 🧾 **Interactive Menu Interface**  
  A user-friendly interface to browse, select, and order food and beverages effortlessly.

## 🛠️ Tech Stack

- **Frontend**: Javascript, HTML, CSS
- **Backend**: Node.js, Express.js  
- **Database**: PostgreSQL  
- **AI Integration**: OpenAI API
